# CharacterCountWinForms

Small Windows Forms application that shows the number of characters in the input text (including or excluding spaces).

## Contents

- CharacterCountWinForms/CharacterCountWinForms.csproj
- CharacterCountWinForms/Program.cs
- CharacterCountWinForms/MainForm.cs
- CharacterCountWinForms/MainForm.Designer.cs

## Requirements

- .NET 7 SDK (or .NET 6/8 with Windows support)
- Windows OS (WinForms requires Windows)

## Build and run (CLI)

1. Clone the repository:
   git clone https://github.com/hakkimakgi-dotcom/MyProject.git
2. Change directory to the project folder:
   cd MyProject/CharacterCountWinForms
3. Build and run with dotnet CLI:
   dotnet build
   dotnet run

## Open in Visual Studio

- Open the CharacterCountWinForms folder or create a new Windows Forms app and add the files.

## Notes

- The app updates character counts in real time and has a checkbox to ignore spaces.
